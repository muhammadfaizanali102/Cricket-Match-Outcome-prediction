# 🏏 Cricket Match Outcome Prediction - AI Project

## 📌 Project Overview

This AI project predicts the outcome (winner) of cricket matches using Machine Learning models.

The system analyzes match features such as:
- Team 1
- Team 2
- Toss Winner
- Toss Decision
- Venue
- Runs
- Wickets
- Overs

It then predicts the match winner using multiple ML algorithms and compares their performance.

## 🎯 Project Domain

**Field:** Sports Analytics

This project applies Artificial Intelligence in sports data analysis to forecast match results.
